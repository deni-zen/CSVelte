Trait HandlesQuotedLineTerminators

Handle line terminators that fall within a quoted string When line terminators fall within a quoted string, they should be fead into the column just like any other character, rather than starting a new row of CSV data. In order for a reader to be able to handle newlines in this way, it must implement this trait.
