Class Stream

CSVelte Stream.

Represents a stream for input/output. Implements both readable and writable interfaces so that it can be passed to either CSVelte\Reader or CSVelte\Writer.

CSVelte\IO\Stream implements CSVelte\Contract\Readable, CSVelte\Contract\Writable, CSVelte\Contract\Seekable uses CSVelte\Traits\IsReadable, CSVelte\Traits\IsWritable, CSVelte\Traits\IsSeekable

Reads all data from the stream into a string, from the beginning to end.

This method MUST attempt to seek to the beginning of the stream before reading data and read the stream until the end is reached.

Warning: This could attempt to load a large amount of data into memory.

This method MUST NOT raise an exception in order to conform with PHP's string casting operations.
