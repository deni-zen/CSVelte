Add path to list of search paths

Add path to list of search paths

Attempts to deduce the absolute (real) path from the path specified by the $path argument. If successful, the absolute path is added to the search path list and the method returns true. If one can't be found, it adds $path to the search path list, as-is and returns false

# register( )

Register the autoloader

Register the autoloader

Registers this library's autoload function with the SPL-provided autoload queue. This allows for CSVelte's autoloader to work its magic without having to worry about interfering with any other autoloader.

Also adds all of this class's search paths to PHP's include path.

Returns

boolean
Whatever the return value of spl_autoload_register is
